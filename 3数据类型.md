+ ES6有6种简单的数据类型：Undefined、Null,Boolean,Number,String,Symbol(ES6新增)
+ 一种复杂的数据类型Object

用typeof操作符包含一个值将会返回7种字符串

+ "undefined" 未定义或者未初始化
+ "boolean" 布尔值

+ "string" 字符串
+ "number" 数值
  + 特殊数值NaN 任何涉及到NaN的运算都会返回false
+ "object" 对象或者Null
+ "function" 函数
+ "symbol" 符号
